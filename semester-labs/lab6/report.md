# Лабораторная работа 6

## Тема
Оптимизация SQL-запросов.

## Цель
Подобрать и обосновать улучшения для 3 запросов из Lab 1/4 и показать эффект.

## Что сделано
Файл [optimization.sql](optimization.sql) содержит:
1. Планы выполнения `EXPLAIN (ANALYZE, BUFFERS)` до оптимизации.
2. Индексы под реальные фильтры и соединения.
3. Переписывание условий (например, диапазон по времени вместо `::date`).
4. Повторные планы выполнения после оптимизации.

## Какие оптимизации применены
1. Индексация по `flights(departure_airport, scheduled_departure)`.
2. Индексация по `semester_ext.loyalty_operations(operation_type, flight_id)`.
3. Индексация по `ticket_flights(amount DESC)`.
4. Локальная переработка запроса с максимумом через `ORDER BY ... LIMIT 1`.
