# Лабораторная работа 1

## Тема
Развертывание и первичный анализ PostgreSQL БД demo.

## Цель
1. Ознакомиться со структурой предметной области авиаперевозок.
2. Подготовить и выполнить набор аналитических запросов.
3. Научиться соединять таблицы, работать с датами, агрегацией и подзапросами.

## Краткое описание выполненного
Подготовлен файл [queries.sql](queries.sql) с 12 запросами по заданию. Запросы используют таблицы `tickets`, `ticket_flights`, `flights`, `boarding_passes`, `airports_data`, `seats`, `aircrafts_data`, `routes`.

## Пояснение по подходу
1. Для поиска маршрутов пассажиров использованы соединения `tickets -> ticket_flights -> flights`.
2. Для анализа расписания использованы фильтры по дате и времени (`::date`, `interval`).
3. Для задач по максимумам/минимумам и подсчетам применены `CTE`, `GROUP BY`, `COUNT`, `MAX`, `MIN`, `SUM`.
4. Для проверки возможности рейса в среду использован `EXISTS` и массив `days_of_week`.

## Результаты
1. Все 12 запросов подготовлены и готовы к запуску в PostgreSQL.
2. Запросы покрывают условия из методички (поиск, агрегация, фильтрация, маршруты).

1. Выполнение 2-3 любых запросов из файла и интерпретацию результата.
2. Самостоятельно написанный дополнительный запрос по выбранной теме.
