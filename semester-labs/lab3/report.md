# Лабораторная работа 3

## Тема
Реализация изменений структуры БД и преобразование данных без потерь.

## Цель
1. Внедрить проект Lab 2 в рабочую БД demo.
2. Выполнить безопасную миграцию данных.
3. Подтвердить корректность результатов проверочными запросами.

## Что выполнено
В файле [migration.sql](migration.sql):
1. Созданы новые таблицы в схеме `semester_ext`.
2. Добавлены индексы на ключевые поля доступа.
3. Созданы аккаунты лояльности по существующим пассажирам.
4. Выполнена первичная загрузка миль по историческим перелетам (`Arrived`).
5. Пересчитан баланс миль по журналу операций.

## Почему данные не теряются
1. Исходные таблицы demo не изменяются и не удаляются.
2. Миграция реализована через `INSERT ... SELECT` в новые таблицы.
3. Скрипт запускается в транзакции (`BEGIN/COMMIT`).
4. Для повторного запуска применены `IF NOT EXISTS` и проверки на дубли.

## Результат
Расширение структуры внедрено, данные сохранены, подготовлена база для следующих лабораторных.
